📦 Installation Guide (Developer Mode)

Follow these steps to install the unpacked Chrome extension:

  1. Open Chrome and navigate to: chrome://extensions/
  2. Enable the "Developer mode" toggle in the top right corner.
  3. Click the "Load unpacked" button.
  4. Select the PolyButton folder containing the manifest.json file.
  5. Click "Select" and you're ready to trade! ✓
Advanced Developer Configuration:

You can customize the button percentages and UI colors by editing content.js:

const CONFIG = {
  primaryColor: '#0393FC',
  percentages: [25, 50, 75, 100],
  checkInterval: 500
};

Frequently Asked Questions

Everything you need to know about the PolyButton extension.

Why aren't the quick bet buttons appearing?

Ensure you are on the official Polymarket website and the extension is enabled. Try refreshing the page. The buttons only appear when a bet panel is open.

Why is nothing happening when I click a percentage?

The Polymarket interface might still be loading, or your wallet balance hasn't synced. Wait a few seconds and try again. You can also check the browser console (F12) for troubleshooting.

Can I change the default percentages (e.g., 10%, 20%)?

Yes! Open the extension folder, edit the content.js file, and modify the percentages array inside the CONFIG object to match your trading strategy.

Is this Polymarket extension safe to use?

100% safe. PolyButton is an open-source frontend script. It only performs local DOM manipulation to inject buttons and calculate math. It does not access, collect, or transmit your private keys, wallet data, or any sensitive information.
